# linenoise

WIP: Bindings for Crystal to the lightwieght Linenoise line editor library.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     linenoise:
       github: your-github-user/linenoise
   ```

2. Run `shards install`
